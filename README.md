# Introduction
I provide my solution to Gilded Rose Refactoring Kata.

# Installation
Clone this repository:
```
git clone https://github.com/ramasofficial/kilo-refactoring.git
```

Install composer:
```
composer install
```

Run tests:
```
composer test
```

Check code:
```
composer check-cs
```
